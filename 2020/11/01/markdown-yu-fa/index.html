Markdown语法

转自菜鸟教程https://www.runoob.com/markdown/md-tutorial.html

标题

使用 # 号可表示 1-6 级标题，一级标题对应一个 # 号，二级标题对应两个 # 号，以此类推。快捷键（ctrl+1.2.3.4.5.6）

Markdown 段落格式

Markdown 段落

Markdown 段落没有特殊的格式，直接编写文字就好，段落的换行是使用两个以上空格加上回车。

当然也可以在段落后面使用一个空行来表示重新开始一个段落。
